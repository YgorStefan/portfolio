**Change 2 — Add flash banners inside the form, immediately after @csrf and before the first field `<div>`.** Insert:

```blade
    {{-- Success banner — CONTACT-04 --}}
    @if(session('success'))
        <div class="bg-green-900/30 border border-green-500/30 text-green-400 rounded-lg px-4 py-3 text-sm">
            {{ session('success') }}
        </div>
    @endif

    {{-- Error banner (mail dispatch failure) — CONTACT-04 --}}
    @if(session('error'))
        <div class="bg-red-900/30 border border-red-500/30 text-red-400 rounded-lg px-4 py-3 text-sm">
            {{ session('error') }}
        </div>
    @endif

    {{-- Validation error summary — a11y fallback --}}
    @if($errors->any())
        <div class="bg-red-900/30 border border-red-500/30 text-red-400 rounded-lg px-4 py-3 text-sm">
            Por favor, corrija os erros abaixo.
        </div>
    @endif
```

**Change 3 — Update the submit button (currently near line 325).** Replace the current button:

```blade
<button type="submit"
        class="w-full bg-accent hover:bg-accent/90 text-white font-semibold
               py-3 px-6 rounded-lg transition-all duration-300 hover:-translate-y-0.5
               focus:outline-none focus:ring-2 focus:ring-accent focus:ring-offset-2 focus:ring-offset-bg-card">
    Enviar Mensagem
</button>
```

With:

```blade
<button type="submit"
        :disabled="submitting"
        class="w-full bg-accent hover:bg-accent/90 text-white font-semibold
               py-3 px-6 rounded-lg transition-all duration-300 hover:-translate-y-0.5
               focus:outline-none focus:ring-2 focus:ring-accent focus:ring-offset-2 focus:ring-offset-bg-card
               disabled:opacity-50 disabled:cursor-not-allowed disabled:hover:translate-y-0"
        x-text="submitting ? 'Enviando...' : 'Enviar Mensagem'">
    Enviar Mensagem
</button>
```

Critical note about Alpine: `@submit="submitting = true"` is on the `<form>` tag (not `@click` on the button). This ensures the form submits first, then Alpine sets the state — using `@click` on the button can block submission in some browsers.
